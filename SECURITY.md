# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this portfolio site or its source code, please report it privately rather than opening a public issue.

- **Contact:** nikunjprajapati09@gmail.com
- Please include a clear description, steps to reproduce, and any relevant URLs or commit references.
- I aim to acknowledge reports within 7 days.

## Scope

This repository hosts a personal portfolio site deployed to GitHub Pages. Reports are welcome for:

- Source code vulnerabilities (XSS, dependency CVEs, exposed secrets)
- Misconfigured headers, CSP, or third-party integrations (Firebase, Google Analytics)

Out of scope: missing rate-limiting on public read endpoints, social-engineering, denial-of-service via traffic volume, and findings that require physical or privileged access.

## Disclosure

Please give a reasonable window to remediate before any public disclosure. Thank you for helping keep this project safe.
